# student-management-system-c
A menu-driven Student Management System built in C featuring CRUD operations, dynamic memory allocation, file persistence, searching, sorting, and input validation.

Overview

A menu-driven Student Management System developed in the C programming language. The project allows users to manage student records through various operations such as adding, displaying, searching, deleting, sorting, saving, and loading student information.

This project was built to practice core C programming concepts including structures, pointers, dynamic memory allocation, file handling, sorting algorithms, and input validation.

---

Features

- Add new student records
- Display all student records
- Search students by name
- Find the topper based on marks
- Delete student records
- Sort students by name
- Sort students by marks
- Save records to a file
- Load records from a file
- Input validation for user entries
